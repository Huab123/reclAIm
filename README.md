# reclAIM

Top-down shooter made in Unity

# Unity running notes
If you want to successfully run the project in Unity do the following:

1. Set up script execution order (First to Last)
-  InventoryDictonary
-  InventoryController
-  SaveController


# Working on Project

1. Set up git in folder
- Create project folder
- `cd {folder_name}`
- `git init`
- `git remote add origin {project ssh link}`
  - Don't forget to setup ssh key 
- `git pull origin main`
- `git push origin master`
- In github double check then pull request from master to main
